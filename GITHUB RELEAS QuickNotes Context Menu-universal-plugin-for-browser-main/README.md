🎯 FocusMode: Productivity Timer Extension
FocusMode — это браузерное расширение (Manifest V3), разработанное для минимизации цифровых отвлечений. Оно позволяет пользователям блокировать доступ к списку социальных сетей и развлекательных ресурсов на заданный промежуток времени, используя высокопроизводительный сетевой фильтр.

🚀 Основной функционал
Динамический «Черный список»: Возможность добавлять и удалять любые домены (например, vk.com, youtube.com) в реальном времени.

Таймер обратного отсчета: Установка времени фокуса (от 1 до 120 минут).

Автоматическая разблокировка: Использование системы будильников браузера (chrome.alarms) для снятия ограничений даже при закрытом окне расширения.

Нативная блокировка: Использование declarativeNetRequest, что гарантирует блокировку сайта до начала загрузки его контента (экономия трафика и ресурсов).

Сохранение состояния: Ваши настройки и список сайтов сохраняются в локальном хранилище браузера (chrome.storage).

Умный интерфейс: Блокировка кнопок управления во время активной фазы фокуса, чтобы избежать искушения удалить сайт из списка.

🛠 Технологический стек
Язык: JavaScript (ES6+), HTML5, CSS3.

Platform: Web Extensions API (Chromium).

Core APIs:

chrome.declarativeNetRequest: Самый современный и безопасный метод блокировки сетевых запросов в Manifest V3.

chrome.alarms: Управление фоновыми событиями и таймерами.

chrome.storage.local: Хранение пользовательских данных.

chrome.scripting & chrome.activeTab: Подготовка к внедрению контента (согласно вашим требованиям к манифесту).

🏗 Архитектура проекта
Проект организован по модульному принципу для обеспечения стабильной работы в фоновом режиме:

Plaintext
focus-mode/
├── manifest.json       # Главный файл конфигурации (права, скрипты, иконки)
├── background.js       # Service Worker: управляет сетевыми правилами и таймером
├── popup.html          # Интерфейс управления (окно по клику на иконку)
├── popup.js            # Логика интерфейса: обработка кликов, работа с хранилищем
└── styles.css          # Стилизация интерфейса расширения
Popup: Служит пультом управления. Взаимодействует с background.js через систему сообщений (chrome.runtime.sendMessage).

Background (Service Worker): «Мозг» расширения. Он не спит, когда работают правила, и просыпается по сигналу alarms, чтобы вернуть доступ к сайтам.

📥 Инструкция по установке (универсальная для всех Chromium-браузеров)
Данная инструкция подходит для Google Chrome, Яндекс.Браузера, Microsoft Edge, Opera и Brave.

Скачайте или создайте файлы: Убедитесь, что все файлы расширения (manifest.json, background.js, и т.д.) находятся в одной папке на вашем компьютере (например, в папке focus-mode).

Откройте страницу расширений:

В адресной строке введите: chrome://extensions/ (или edge://extensions/ для Edge).

Включите «Режим разработчика»:

Найдите переключатель Developer mode (Режим разработчика) в правом верхнем углу и активируйте его.

Загрузите расширение:

Нажмите на кнопку Load unpacked (Загрузить распакованное расширение).

В появившемся окне выберите папку focus-mode, в которой лежат ваши файлы.

Закрепите расширение:

Нажмите на иконку «пазла» в панели инструментов браузера и закрепите (pin) Focus Mode, чтобы таймер всегда был под рукой.

⚠️ Важное примечание
Для корректной работы блокировки в Manifest V3, убедитесь, что при вводе сайта в список вы указываете чистый домен (например, facebook.com), так как фильтр настроен на поиск этого вхождения в URL.
