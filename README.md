# ВНИМАНИЕ: ЭТО АЛЬФА-ДЕМО-ТЕСТОВОЕ РАСШИРЕНИЕ ДЛЯ ПОРТФОЛИО, ФУНКЦИОНАЛ МОЖЕТ БЫТЬ НЕПОЛНЫМ ИЛИ НЕДОРАБОТАННЫМ

# 📝 QuickNotes Context Menu: Smart Web Clipper

[![Manifest V3](https://img.shields.io/badge/Manifest-V3-orange.svg)](https://developer.chrome.com/docs/extensions/mv3/intro/)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

**QuickNotes Context Menu** — это легкое и эффективное расширение для браузеров на базе Chromium, которое позволяет мгновенно сохранять важную информацию из сети. Забудьте о переключении вкладок между браузером и блокнотом.

---

## ✨ Основные возможности

* **🖱 Интеграция в контекстное меню**: Выделите любой текст на странице -> Правая кнопка мыши -> "Сохранить в заметки".
* **✏️ Живое редактирование**: Изменяйте текст сохраненных заметок прямо в интерфейсе расширения.
* **📂 Экспорт в .TXT**: Скачивайте весь список заметок одним файлом с сохранением ссылок на источники.
* **🔢 Умная сортировка**: Переключайте порядок отображения (новые сверху или старые сверху) без потери данных.
* **🔗 Сохранение метаданных**: Расширение автоматически запоминает дату создания и URL-адрес страницы, с которой был взят текст.
* **🛡 Безопасность**: Все данные хранятся локально в вашем браузере (`chrome.storage.local`), не отправляясь на сторонние серверы.

## 🚀 Инструкция по установке

Подходит для **Google Chrome, Microsoft Edge, Brave, Opera и Яндекс.Браузера**.

1.  **Скачайте проект** 📥
    * Клонируйте репозиторий или скачайте архив с файлами расширения. Распакуйте в отдельную папку.
2.  **Перейдите в раздел расширений** 🌐
    * Вставьте в адресную строку: `chrome://extensions/` (или `edge://extensions/`).
3.  **Активируйте режим разработчика** 🛠️
    * Включите тумблер **Developer mode** в правом верхнем углу.
4.  **Загрузите папку** 📤
    * Нажмите **Load unpacked** (Загрузить распакованное расширение) и выберите папку с проектом.
5.  **Начните работу** 📌
    * Закрепите иконку расширения в панели инструментов для быстрого доступа к списку заметок.
